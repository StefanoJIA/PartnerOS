# D5.18 Soft Quote Handoff Workspace

## Goal

在不创建正式 quote 的前提下，把产品机会与 pre-quote 信息整理成内部报价准备交接包。

## Business Focus

- HOSUN lifting systems
- adjustable desk frames
- desk legs
- lifting columns
- heavy-duty lifting systems
- JOOBOO education furniture
- medical / lab workspace
- project supply
- OEM / ODM components

## Workflow

1. 打开 `/lead-intelligence`。
2. 选择 lead。
3. 查看 Product Fit / Pre-Quote Prep。
4. 查看 **Soft Quote Handoff**。
5. Copy Customer Questions / Supplier Notes / Handoff Brief。
6. 人工确认缺失信息。
7. 后续由正式 Quote 模块承接。

## Safety

- no formal quote created
- no pricing generated
- no inventory promise
- no certification promise
- no lead-time promise
- no automatic sending

## Acceptance Criteria

- quote handoff brief 可用
- customer questions 可复制
- supplier notes 可复制
- tests pass
- no DB migration

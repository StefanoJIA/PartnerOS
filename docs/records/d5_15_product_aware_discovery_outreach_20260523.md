# D5.15 Product-Aware Discovery Outreach Bridge

## Goal

把 product fit、quote readiness、pre-quote brief 转化为人工审核的产品导向 discovery draft。

## Business Focus

- HOSUN lifting systems
- adjustable desk frames
- desk legs
- lifting columns
- JOOBOO education furniture
- medical / lab workspace
- project supply
- OEM / ODM components

## Workflow

1. 打开 `/lead-intelligence`。
2. 选择 lead。
3. 查看 Product Fit / Pre-Quote Prep。
4. 打开 Product-Aware Draft。
5. 选择 channel 与 purpose。
6. Generate Draft → Copy → 人工审核 → 外部发送。
7. Mark as Sent → Set follow-up。

## Safety

- no automatic sending
- no quote creation
- no pricing
- no inventory promise
- no certification promise
- no lead-time promise

## Acceptance Criteria

- product-aware draft 可用
- LinkedIn note <= 300 chars
- copy 可用
- forbidden promise 检查通过
- tests pass
- no DB migration
